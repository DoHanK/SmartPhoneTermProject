# **인버서스**
---
   ### **1.게임 컨셉**
 ---
 > #### High Concept: 2D 슈팅 액션 게임
 > #### 핵심 메카닉 간단한 규칙과 간단한 조작을 통해 다가오는 적을 섬멸하고 살아남는 게임


 ### **2.개발 범위**
 ---
 >   #### **게임 규모**
 >>  ##### **1.맵**
 >>> ###### 1개(셀 50 X 50 사이즈)

 >>  ##### **2.캐릭터** 
 >>> ###### **크기** : 1셀 크기
 >>> ###### **총알** : 최대총알8알
 >>>> ###### **총알발사쿨타임** :0.1초
 >>>> ######  **한발당 데미지** : 적 사살, 벽 뿌시기

 >> ##### **3.적**
 >>> ###### **크기:** 1셀 크기
 >>> ###### **특징:** 지나간 길은 플레이어가 지나갈수없는 벽으로 만듬 

>> ##### **4.게임 최종 목적**
>>> ###### **최고 점수 만들기 (적 한명당 score 50추가, 1초 이내에 사살시 콤보 점수 부여)**

 ### **3.예상 게임 실행 흐름**
 ---
 > #### **로비 -> 게임 플레이 -> 종료 OR 로비로 이동**
